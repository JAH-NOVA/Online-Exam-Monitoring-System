 # 🎯 Online Exam Monitoring System

## 📌 Overview
This project is designed to detect cheating behavior during online examinations using Computer Vision and Machine Learning techniques. It monitors suspicious activities such as head movement, multiple persons, and unusual behavior.

## 🚀 Features
- Real-time monitoring using webcam
- Detects suspicious head movements
- Alerts for multiple person detection
- Logs and stores suspicious activities

## 🛠️ Technologies Used
- Python
- OpenCV
- Machine Learning
- Flask (for web interface)

## 📂 Project Structure
- detectors/ → detection logic
- templates/ → frontend UI
- static/ → CSS & JS files
- app.py → main application

## ▶️ How to Run
1. Install dependencies  
pip install -r requirements.txt
2. Run the app
python app.py
3. Open browser and access the system
